# assets/emoticons/

Coloque aqui os **emoticons retrô** clássicos do MSN (GIFs/PNGs pixelados).

## Sugestões de arquivos

| Arquivo         | Atalho de texto | Descrição                |
|-----------------|-----------------|--------------------------|
| `smile.gif`     | `:)`            | Sorriso                  |
| `sad.gif`       | `:(`            | Triste                   |
| `tongue.gif`    | `:P`            | Língua de fora           |
| `wink.gif`      | `;)`            | Piscadinha               |
| `devil.gif`     | `(6)`           | Diabinho                 |
| `heart.gif`     | `(L)`           | Coração                  |
| `nudge.gif`     | —               | Ícone do botão de Nudge  |

Cada emoticon é mapeado ao seu atalho de texto na janela de conversa.
