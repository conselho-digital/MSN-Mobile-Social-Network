# assets/avatars/

Fotos de exibição **de exemplo** — uma galeria de opções prontas que a
pessoa pode escolher como foto de perfil, sem precisar enviar uma foto
própria (igual às figurinhas padrão do MSN clássico).

## Como enviar

Envie os arquivos aqui com nomes sequenciais, por exemplo:

```
avatar1.webp
avatar2.webp
avatar3.webp
...
```

Formato recomendado: **WebP com fundo transparente ou quadrado**
(a imagem é cortada com `cover`, então quanto mais próxima de 1:1,
melhor).
