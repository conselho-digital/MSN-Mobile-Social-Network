# assets/scenes/

Imagens de **cenário** — o fundo do topo do Dashboard (do cabeçalho do
perfil até a barra "Pesquisar contatos"). Da barra de pesquisa para
baixo, a cor segue o **tema** pareado ao cenário (isso não muda aqui).

## Como enviar

Cada cenário já cadastrado no site (`js/scenes.js`) espera uma imagem
com o **nome exato** abaixo. Envie um arquivo `.jpg` com esse nome para
substituir automaticamente o degradê atual pela sua imagem — não
precisa mexer em código.

| Arquivo         | Cenário       |
|-----------------|---------------|
| `green.jpg`     | Verde         |
| `blue.jpg`      | Azul          |
| `aero.jpg`      | Aero          |
| `purple.jpg`    | Roxo          |
| `pink.jpg`      | Rosa          |
| `sunset.jpg`    | Pôr do sol    |
| `teal.jpg`      | Turquesa      |
| `graphite.jpg`  | Grafite       |
| `royal.jpg`     | Royal         |

## Dimensões recomendadas

- Proporção próxima de **16:9** (ex.: 800×450px), a imagem é cortada
  com `cover` (preenche o espaço, sem distorcer).
- Se o arquivo de um cenário não existir, o degradê de cor atual
  continua aparecendo no lugar (nada quebra).

## Adicionar novos cenários (além dos 9 atuais)

Quer mais opções na grade (como na tela clássica do Messenger)? Me
avise quais imagens você quer adicionar — cada cenário novo precisa
de um nome, uma cor de tema pareada e o arquivo de imagem.
