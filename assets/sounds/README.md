# assets/sounds/

Coloque aqui os **efeitos sonoros clássicos** do MSN (formato `.mp3`).

## Arquivos esperados

| Arquivo         | Uso                                                    |
|-----------------|--------------------------------------------------------|
| `login.mp3`     | Som de entrada/conexão (o clássico "da-na-na-naaa")    |
| `message.mp3`   | Som de mensagem recebida ("tucutucu")                  |
| `nudge.mp3`     | Som da "Chamada de Atenção" / buzzer (tremida de tela) |
| `logout.mp3`    | Som de contato saindo (opcional)                       |

Estes nomes são usados pelo `js/sound-manager.js`.
