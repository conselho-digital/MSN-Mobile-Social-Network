# Sons

Coloque aqui os 4 arquivos abaixo, com **esse nome exato** e em
**formato `.mp3`** (o navegador não toca `.wma`):

| Arquivo         | Quando toca                              |
|------------------|-------------------------------------------|
| `login.mp3`      | Um contato silenciado* fica online        |
| `message.mp3`    | Chega uma mensagem nova                   |
| `nudge.mp3`      | Alguém te chama a atenção ("Nudge")       |
| `logout.mp3`     | Você sai da conta                         |

\* não toca se você silenciou as notificações desse contato (menu de
segurar apertado na lista de contatos).

Veja `js/sound-manager.js` — os nomes/caminhos precisam bater
exatamente com essa tabela.
